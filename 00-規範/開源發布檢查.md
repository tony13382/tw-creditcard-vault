# 開源發布檢查

這個 vault 的遠端 repository 只用於公開信用卡資料。推送前必須確認沒有個人持卡、額度、帳戶或推薦結果。

## 必須排除

- `40-私有/`
- `.obsidian/`
- 本機匯出、暫存、同步衝突檔。
- 任何個人信用額度、持卡清單、帳戶等級、會員狀態、自扣、電子帳單、登錄、領券、App 方案、申辦日、核卡日、帳單日、繳款日。

## 檢查命令

```sh
git status --short --ignored
rg -n "40-私有|目前信用額度|可用額度|持有中($|[[:space:]，。；、])|額度來源：使用者|使用者提供|本次提供|目前已記錄|核卡日期[:：]|帳單日[:：]|繳款截止[:：]|自動扣繳：未記錄|信用額度 (80,000|150,000|160,000|250,000)|額度 (80,000|150,000|160,000|250,000)" CLAUDE.md index.md log 10-來源 20-Wiki 30-研究 觀察清單.md
rg -n "\[\[40-私有|40-私有/" CLAUDE.md index.md log 10-來源 20-Wiki 30-研究 觀察清單.md
```

## 允許保存

- 官方條款中的公開信用額度公式。
- 官方要求的公開條件，例如需電子帳單、需自動扣繳、需會員等級。
- 不指向特定個人的研究狀態與資料缺口。
