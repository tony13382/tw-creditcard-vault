# 中國信託財管鼎鑽卡

## 定位

中國信託**財富管理會員專屬**信用卡。發卡資格綁定本行個人財富管理會員等級（首富家 / 鼎富家 / 傳富家 / 臻富家）；非財管會員無法申辦。

核心結構：**免切換國外最高 3%、6 次彈性交通禮遇（可在高鐵 / 機場接送 / 貴賓室間互換）、4,200 家全球奢華酒店訂房禮遇、米其林餐廳優先預訂、24 小時生活管家、5,000 萬旅平險 + 海外醫療救助 319 萬。**

> 入口：https://mkt.ctbcbank.com/long/creditcard/WMmember/index.html

## 排除條款

- 預借現金、特約商店分期、稅款、規費、政府機關費用、第三方支付（依官方公告）通常不享回饋。
- 各加碼活動依官方公告之有效期間與條件為準，過期後須複查。
- 退貨或爭議款將扣回相應回饋。

## 年費

- 官方產品頁未直接揭露年費實額；財管專屬卡通常會搭配財管會員身份免年費或減免，需另查本行公告。
- 信用卡循環年利率：ARMs 指數 + 5.97% ～ 13.47%（上限 15%）。
- 預借現金手續費：每筆預借金 × 3.5% + NT$150。

## 適合情境

- 已是本行財管會員（鼎富家以上），且想要：
  - 國外消費免切換 3% 無上限。
  - 6 次彈性交通禮遇（高鐵 + 機場接送 + 貴賓室）。
  - 全球 4,200 家奢華酒店訂房禮遇（提前入住、延後退房、房型升等等）。
  - 米其林餐廳優先預訂。
  - 高奢酒店住三付二（鼎富家以上）。
  - 環宇尊榮通關買一送一（鼎富家以上）。
  - 5,000 萬旅平險 + 319 萬海外醫療救助。
- 想用財管會員身份取代鼎極卡 5,000 元年費的高階卡需求。

## 不適合情境

- 非本行財管會員：無法申辦。
- 主力為國內現金回饋 / 點數系：1% / 0.8% 不具優勢。
- 部分歐洲國家實體消費：本卡無回饋。
- 不在意機場接送 / 米其林 / 酒店禮遇者：高階禮遇用不到。
