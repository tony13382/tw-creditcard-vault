# 國泰世華 CUBE 卡

## 定位

權益方案切換型**小樹點（信用卡）回饋卡**。CUBE 卡的推薦判斷要同時看三件事：**當日切換的權益方案**、**CUBE 權益等級**、**該筆消費是否符合指定通路與排除條款**。

> 持卡人歸戶限申辦 **1 張** CUBE 卡。
> 自 2026/1/1 起每月指定消費上限：**消費日當日信用額度 + NT$50 萬**；正、附卡信用額度與溢繳金額合併計算。

## 7 大權益方案（2026 通用）

CUBE 卡提供 7 大權益方案，**每正卡人每日可免費變更 1 次**（適用 2026/6/30 前），附卡適用方案依正卡持卡人設定。方案設定**自當日 0 時起**之消費即依當次變更後方案計算。

| 方案 | 通路類型 |
|---|---|
| **玩數位** | 指定線上消費（網購平台、Apple 媒體服務等） |
| **樂饗購** | 指定美食 / 購物（百貨、外送平台、國內餐飲、誠品生活、康是美、屈臣氏等） |
| **趣旅行** | 指定旅遊 / 交通（航空、訂房平台、計程車、租車等） |
| **集精選** | 指定品牌 / 生活（量販超市 + 便利商店 + 日常出行 + 生活家居，含 OPEN 錢包 / My FamiPay / PX Pay 店內消費） |
| **慶生月** | 當月壽星專屬 |
| **童樂匯** | 親子專屬 |
| **JCB 日本賞** | JCB 卡別專屬 |

## 使用條件

### 信用額度上限

- 自 2026/1/1 起，CUBE 信用卡每月指定消費之累計消費金額以「消費日當日信用額度」加計新臺幣 **50 萬**為限。
- 信用額度異動時，調整後金額上限不溯及適用至當月已核發之小樹點消費；超出限額之消費僅適用一般消費 0.3% 回饋。
- 正、附卡信用額度與溢繳金額合併計算。
- 持卡人當月回饋之小樹點（信用卡）總額，以本行最終檢核之結果為準；本行得逕行扣回逾發之小樹點。

### 方案切換

- 需於 **CUBE App** 或官方指定方式切換權益方案；2026/6/30 前每正卡人每日可免費變更 1 次。
- 方案設定自當日 0 時起之消費即依當次變更後方案計算，依本行系統紀錄為準。
- 舉例：小華 2026/1/10 17:00 變更方案為「樂饗購」，且當日 10:30 / 18:00 各有一筆消費 → 整日所有消費（包含 10:30 那筆）皆依「樂饗購」方案計算回饋。

### 支付方式（指定消費 / 優惠券限定）

- 各權益方案之「指定消費」或「優惠券」消費，**限直接刷 CUBE 卡或加入 Apple Pay / Samsung Pay / Google Pay / Garmin Pay**。
- **不含**透過綠界支付平台、藍新支付平台、91APP 或其它第三方支付平台之交易。
- **不含**透過綁定其他型態支付工具之消費：LINE Pay、街口支付、全盈 + PAY、全支付、Pi 錢包、橘子支付、歐付寶、icash pay、OPEN 錢包（除集精選方案 OPEN 錢包於超商消費例外）、My FamiPay（除集精選方案外）、悠遊付、ezPay、GOMAJI Pay、friday wallet、PayPal、支付寶等。

### 入點與折抵

- 每筆消費請款入帳後，依**授權日**判斷適用權益方案、符合之優惠券，以各別回饋率計算小樹點。
- 入點：次 2 日（遇假日順延）回饋至正卡人信用卡帳戶。
- 消費折抵金：於每筆消費請款入帳後計算，次 3 日（遇假日順延）回饋至正卡人信用卡帳戶，系統自動將該筆回饋金額折抵於原刷卡交易。
- 正附卡消費合併計算，小樹點回饋歸戶至正卡持卡人信用卡帳戶。

### 退貨扣回

- 退貨將比對該筆退貨交易原始消費之適用權益方案，依該方案計算回饋後予以扣除。
- 若商家退款交易資料不足以辨認原始適用方案，則依一般消費計算扣除。
- 若點數扣回時持卡人已將點數使用完畢，改以 1 點 = NT$1 入信用卡帳單。

## 排除條款

### 一般消費 0.3% 與指定消費回饋皆排除

- 信用卡相關利息及費用（年費、掛失費、預借現金手續費、違約金、國外交易手續費、調閱簽帳單費、補發帳單費等）。
- 預借現金、帳單分期、單筆消費分期（**刷卡樂分期**，自 2024/2/1 起於特約商店分期付款消費含提前結清，僅適用 0.3%）。
- 通信貸款、餘額代償金額及經本行公告除外之專案。
- 稅款、學費、公用事業費用（含網路通訊、影音服務等相關費用及附加費）。
- 路邊停車費、公務機關各項費用。
- 透過繳費平台（公務機關信用卡繳費平台、電子化繳費稅、醫指付、CUBE 網銀 / App 線上繳費）繳納之費用。
- 代繳國泰人壽**躉繳、投資型及利率變動型年金**保單；代扣基金。
- 悠遊卡 / 一卡通 / icash 加值。
- 便利超商消費（**集精選方案 7-11 / 全家除外**）；透過聯合信用卡處理中心小額支付平台之特約商店消費（含便利超商、連鎖速食店、停車費等）。
- **全聯福利中心消費（集精選方案除外）/ 大全聯消費**（含實體門市與線上通路）。
- eTag 儲值 / eTag 智慧停車費代扣繳。
- 遠東 SOGO 百貨貴賓廳之消費金額。
- 國壽不動產租金代收。
- 未經主管機關核准之境外投資交易平台（含 eToro）、匯兌平台（含 Wise）、數位（虛擬）貨幣及旅行支票（含 MCC 4829 / 6012 / 6211 / 6051）、博奕（含 MCC 7995）。

- 壽險保費**不含**國泰人壽躉繳、投資型及利率變動型年金保費。

## 附加功能

- **CUBE App 查卡號**：核卡通知後透過 App 取得卡片資訊，可進行 3D 驗證線上刷卡、綁定行動支付、由本行代為綁定聯名商戶會員帳號。
- **COMBO**：信用卡 + 金融卡雙卡合一。
- **儲值卡**：搭配悠遊卡或 icash 卡。

## 適合情境

- 願意在消費前**切換權益方案**。
- 願意用國泰世華**本人帳戶自扣**或 **CUBE App 繳費**取得 Level 2 升級 3% 回饋。
- 消費集中在 CUBE 官方指定通路，且可避開排除項目。
- 持有財管會員身份（資產 NT$300 萬以上）可升 Level 3：最高 3.3%。
- 想累積可即時折抵的小樹點（1:1 抵帳單）。
- 高頻 7-11 / 全家 / 全聯 / 家樂福 / LOPIA / 中油 / IKEA / EV 充電：選定「集精選」方案享 2%。
- 當月壽星：踩點吉章 + 慶生月方案。

## 不適合情境

- 不想管理權益方案切換。
- 沒有國泰世華帳戶自扣或 CUBE App 繳費：只能停留在 Level 1（指定消費 2%）。
- 主要消費是稅費、學費、繳費平台、便利商店（非集精選）、全聯（非集精選）/大全聯、電子票證加值、分期或保費。
- 主要使用第三方支付（LINE Pay、街口、悠遊付等綁定支付）：指定消費不適用。
- 大全聯消費：完全不適用。
