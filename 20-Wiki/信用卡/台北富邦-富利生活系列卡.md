# 台北富邦富利生活系列卡

## 定位

台北富邦**生活通路紅利卡**：8 大生活通路（百貨 / 量販 / 超市 / 餐廳 / 加油 / 書店 / 旅行社 / 航空公司）享 **5 倍紅利**回饋（含原本 1 倍，即 **NT$20 = 5 點**），**回饋無上限**。可選**悠遊卡 / 一卡通**功能；**icash 聯名卡 2026/6/30 終止合約**。

> 本卡為「富利生活系列」，含**富利生活卡**（純信用卡）+ **富利生活悠遊聯名卡** + **富利生活 icash 聯名卡**（停發中）。

| 通路類別 | 代表品牌 |
|---|---|
| **百貨** | 新光三越、遠東 SOGO、微風廣場、遠東百貨 |
| **量販** | 家樂福、大潤發、大買家、愛買 |
| **超市** | 楓康、喜互惠、Jasons |
| **餐廳** | 王品、欣葉、瓦城、星巴克、Friday's |

| **航空公司** | 長榮、中華、國泰、Jetstar、AirAsia |

**累積率：NT$20 = 5 點**（含原本 1 倍），等效約**現金回饋率依紅利兌換率而定**。

> 紅利點數兌換規則依本行公告；常見兌換為刷卡金或航空哩程（轉換率約 NT$25 = 1 點 → 25 點折 NT$1，即 5 倍 = NT$20 累積 5 點 → 折 NT$0.2 ≈ **1% 等效**）。

### 免年費辦法

- **首年免年費**。
- 次年：正附卡合併消費達 **NT$60,000** 或 **12 次**。

### 2026/12/31 前同享免年費條件（任一）

1. 僅申請電子帳單。
2. **綁定銀行 LINE 官方帳號好友**。
3. 本行存戶自扣。

- 退貨或爭議款將扣回相應回饋。

## 適合情境

- 經常於 8 大生活通路消費者，且通路橫跨多類（百貨 + 餐廳 + 加油 + 航空等）：5 倍紅利無上限。
- 想要附**悠遊卡**功能的生活卡：悠遊聯名卡。
- 想累積紅利兌換航空哩程或刷卡金者。
- 不熟悉切換型卡（如 CUBE）的卡友：本卡免切換、不限金額。

## 不適合情境

- 主力消費為非 8 大通路（如電商、行動支付綁卡、保費）：紅利累積率低。
- 想要現金回饋而非紅利：紅利兌換刷卡金等效率不高。
- 想要 icash 功能：自 2026/7/1 起新發卡不再具 icash 功能。
- 7-Eleven / 全家便利商店：不在 8 大通路內，無 5 倍紅利。
