# 星展 HAPPY GO 聯名卡

## 定位

星展與**鼎鼎聯合行銷（HAPPY GO）**聯名信用卡。**HAPPY GO 點數累積卡**：

- 一般消費 **NT$100 = 1 點 HAPPY GO**（基本累積率約 1%）。
- HAPPY GO 特約商店一般消費 **2 倍**點數。
- **遠東 SOGO / 遠百 / 大遠百 / 巨城 / 愛買 / city'super / friDay / 遠企**等遠東通路一般刷卡 **3 倍**點數。
- **HAPPY GO Pay 綁定本卡再享 3 倍**點數（合計**最高 6 倍**）。

> **本聯名卡合作有效期限至 2026/12/31 止**。
> 已持星展 HAPPY GO 聯名卡正卡者不得重複申請。

## 2026 可用回饋（HAPPY GO 點數）

### 點數累積結構

| 通路 | 累積率 | 說明 |
|---|---|---|
| **一般消費** | NT$100 = **1 點** | 約 1% |
| **HAPPY GO 特約商店** | **2 倍** | 原 1 倍 + 特約商店 1 倍 |
| **遠東通路一般刷卡** | **3 倍** | 原 1 倍 + 特約商店 1 倍 + 加碼 1 倍 |
| **遠東通路 + HAPPY GO Pay 綁定** | **6 倍**（3 倍 + 3 倍） | 原一般 + 特約 + 加碼 1 倍 + HAPPY GO Pay 3 倍 |

### 點數入帳機制

| 點數來源 | 入帳方式 |
|---|---|
| 一般消費 1 倍 | 本行入點（最快於每月帳單結帳日轉入 HAPPY GO 帳戶） |
| 加碼 1 倍（含特約商店、遠東通路加碼） | **由鼎鼎聯合行銷直接入點**至 HAPPY GO 帳戶 |
| HAPPY GO Pay 加碼 3 倍 | **由鼎鼎聯合行銷直接入點**至 HAPPY GO 帳戶 |

### 行動支付綁卡規則

- **必須將實體卡交給商家過卡**（包含 Apple Pay、Samsung Pay、LINE Pay、街口支付），方可累積由鼎鼎聯合行銷提供的 1 倍點數。
- 透過 LINE Pay / 街口支付等綁定支付：**1 倍由本行回饋不適用**（依國際組織 MCC 認定）。

## 新戶禮（依本行公告）

- 新戶申辦正卡，核卡日後 30 天內：
  1. 完成註冊登入星展 Card+ 信用卡數位服務。
  2. 成功申請信用卡電子帳單。
  3. 單筆一般消費達 NT$1,000（含）以上。
- 享 **750 點 HAPPY GO 點數**。

## H1 限時加碼（2026/5/7 ～ 2026/6/30）

- 持星展 HAPPY GO 聯名卡於**遠東百貨、遠東 SOGO、遠東巨城購物中心、TheMall 遠企購物中心、遠傳電信、FriDay、Garden City、愛買**等指定通路。
- 累積消費滿 **NT$5,000** → 加贈 **300 點 HAPPY GO 點數**。

## 點數規則

- **點數有效期限**：當年度獲贈之點數可使用至**次年 12/31** 止。
- 正附卡刷卡消費合併計算至正卡持卡人之 HAPPY GO 帳戶。
- 退貨或取消交易：依入帳日回饋比例**自動扣回 HAPPY GO 點數**。
- 持卡人**不適用於遠東 SOGO 卡友禮贈品及來店禮活動**。
- 持卡人喪失資格：停用卡片、掛失超過 3 個月未補發、延遲繳款 / 未繳款未繳最低應繳款者，將喪失參加 HAPPY GO 點數活動之資格，未回饋之點數將自動失效。

## 適合情境

- **遠東集團通路常客**（遠東 SOGO / 大遠百 / 巨城 / 遠百 / 愛買 / friDay）：3 倍累積 + HAPPY GO Pay 6 倍。
- 想要 HAPPY GO 點數體系：可在指定特約商店折抵消費或兌換商品。
- 已是 HAPPY GO 會員：自動合併點數至會員帳戶。

## 不適合情境

- 不去遠東通路：基本累積率僅 1%。
- 想要現金回饋：本卡點數需於 HAPPY GO 體系折抵。
- 主力為 LINE Pay / 街口支付綁卡：1 倍點數無法累積。
- 想累積點數超過 2 年：當年度點數有效期僅至次年 12/31。
- **重要：合作有效期限 2026/12/31** 屆滿後續展與否需另查。
