# 聯邦賴點卡

## 定位

LINE Pay + LINE POINTS 回饋卡。推薦核心是「綁定 LINE Pay + 電子化帳單 + 聯邦帳戶自扣 + 韓國指定店家旅遊」。國內 LINE Pay 最高 **11%**、國外 3% 無上限、**歐洲實體店面 3% 無上限**（與 M 卡相比歐洲未排除為其差異化優勢）。

## 使用條件

### 國內 LINE POINTS 11% 任務

需同時滿足：
1. **申辦電子化帳單**（Email 帳單 / 簡訊帳單 / LINE 帳單）。
2. **綁定 LINE Pay**（聯邦賴點卡綁定台灣 LINE Pay 帳號）。
3. **聯邦銀行帳戶自動扣繳**信用卡款（含 NewNewBank 數位帳戶）。

### LINE Pay 綁卡步驟

1. 開啟 LINE APP → 錢包 → LINE Pay。
2. 同意條款並設定密碼。
3. 註冊卡片：掃描或輸入卡片資料綁定聯邦賴點卡。
4. 我的條碼即可付款。

### LINE POINTS 點數規則

- 限**透過 LINE Pay 使用聯邦賴點卡付款**才可享 LINE POINTS 回饋（無透過 LINE Pay 之實體刷卡僅享基本回饋）。
- 取消交易後已獲得的 LINE POINTS 會回收。
- LINE POINTS 點數有效期依 LINE 規定。
- 部分交易（保費、稅費等）不享 LINE POINTS。

## 排除條款

- 不透過 LINE Pay 付款不享 LINE POINTS 回饋。
- 同一張賴點卡**不可綁定於多個 LINE Pay 帳號**。
- 預借現金、保費（部分）、稅費、繳費平台、特約商店分期等通常不享回饋。
- 取消交易、退貨：LINE POINTS 將被回收。
- 韓國指定店家活動需確認商店明細為指定店家。

## 旅遊與生活權益（賴點一卡通御璽卡）

- **旅遊平安險**：刷付 80% 以上團費或全額交通工具費用 → 享 **NT$2,000 萬旅遊平安險**保障。
- **道路救援**：達指定消費金額享免費 **50 公里**道路救援。
- **機場外圍停車**：免費。
- **國賓影城**：以賴點卡購票享全票 **65 折起**。
- **御璽卡機場接送、Visa Best Offer**、訂房網優惠等。

## 與 M 卡的差異化

| 維度 | 聯邦賴點卡 | 聯邦 M 卡 |
|---|---|---|
| 核心通路 | LINE Pay 綁卡 | 旅遊交通五大場景 |
| 國內最高回饋 | **11% LINE POINTS**（任務制）| 11% 刷卡金（跨場景）|
| 國外消費 | 3% LINE POINTS | 3% 刷卡金 |
| **歐洲實體** | **3% 無上限** | 模糊（部分公告排除）|
| 回饋形式 | LINE POINTS（LINE 生態系折抵）| 刷卡金（直接折抵帳單）|
| 韓國指定店家 | **最高 30%**（特殊優勢）| 無 |
| 保費回饋 | 通常排除 | 1.2%（可選分期）|

## 適合情境

- 重度 LINE Pay 用戶（每月主要消費透過 LINE Pay 綁賴點卡）。
- 有歐洲實體旅遊需求（**3% 無上限**，比 M 卡更穩定）。
- 韓國旅遊（2026 Q2 最高 30%）。
- 願意設定電子化帳單 + 聯邦帳戶自扣解鎖任務。
- 想累積 LINE POINTS 用於 LINE 生態系折抵（LINE 購物、LINE 禮物、LINE TODAY 等）。

## 不適合情境

- 不用 LINE Pay 或不想綁定 LINE 帳號。
- 想要直接折抵信用卡帳單（LINE POINTS 不能直接折信用卡帳單）。
- 主要消費為保費、稅費、繳費平台等排除項目。
- 不想設定電子化帳單與自扣任務。
