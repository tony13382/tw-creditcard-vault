# 國泰世華蝦皮購物聯名卡

## 定位

國泰世華與**蝦皮購物**聯名 **Mastercard 鈦金卡**。**蝦幣回饋（1 蝦幣 = NT$1）**：蝦皮購物站內最高 **10%**（4% 天天 + 6% 指定期間）、指定外送 / 交通 / 旅遊 / 航空 / 旅行社 / 旅遊平台 / 免稅店最高 **7%**、海外消費**免手續費** + 0.5% 蝦幣、月月滿額享免運券。

> **無附卡**。每蝦皮會員帳號限申辦 1 張；持卡人歸戶限申辦 1 張。

## 2026 H1 可用回饋（蝦幣，1 蝦幣 = NT$1）

### 蝦皮購物站內

| 情境 | 蝦幣回饋 | 上限 |
|---|---:|---|
| 全站消費（天天） | **最高 4%** 蝦幣 | 依本行公告 |
| 指定期間再加碼 | **最高 6%** 蝦幣 | 需登錄 |
| **單筆消費 ≥ NT$6,500**（蝦皮購物） | 贈 **650 蝦幣**（≈ 10%） | 需登錄 |
| **單筆且須分期 ≥ NT$15,000** | 贈 **1,800 蝦幣**（≈ 12%） | 需登錄 |
| 蝦皮商城（25 號狂購節） | 單筆 ≥ NT$3,000 享 9 折 | 限量 |

### 站外消費

| 情境 | 蝦幣回饋 |
|---|---:|
| 一般消費 | **0.5%** |
| **指定外送 / 交通 / 航空 / 旅行社 / 旅遊平台 / 免稅店** | **最高 7%**（需登錄） |
| 海外消費 | 0.5% + **免國外交易手續費** |

### 蝦皮購物促刷加碼（2026/1/1 ～ 2026/6/30）

- 指定期間於蝦皮購物刷聯名卡不限消費金額，登錄成功加碼 **2% 蝦幣**回饋。

## 月月 0 元免運券

- 每月帳單蝦皮聯名卡一般消費累積 **5 筆**或 **NT$5,000** → 次月享 **1 張蝦皮店到店免運券**回饋。

## 新戶禮（2026/1/1 ～ 2026/6/30）

- 首次申辦蝦皮購物聯名卡，核卡後 **30 日內**累積一般消費達 **NT$1,000** → 享 **200 蝦幣**。

## 蝦幣折抵

- 蝦皮購物全站消費可用蝦幣折抵消費金額。
- 加碼商城 **88 折**折價券（限量）。
- 加碼全站 **9 折**折價券（限量）。

## 使用條件

- 申辦資格：成年人；現職滿 1 年；年收入 NT$20 萬以上。
- **持卡人須先至蝦皮購物申辦會員**；於蝦皮購物留存之手機號碼需與本行相同。
- 每蝦皮會員帳號限申辦 1 張；持卡人歸戶限申辦 1 張。
- **無附卡**。
- 國泰世華官網申辦：核發信用碼 + 實體卡；紙本申請：核發實體卡。
- 開卡即綁卡：將聯名卡綁定至持卡人申辦時填寫的蝦皮帳戶。

## 排除條款

- 免年費門檻消費計算**不含**：各類預借現金、代償、代收費用、各項作業 / 手續費、退款 / 退貨、掛失費、製卡費、年費、循環利息、服務成本、違約金、爭議款等非消費性款項。
- 各類加碼活動需登錄，且依各活動公告之限額 / 限量 / 期間規範。

## 年費

- 鈦金卡：**NT$1,800**。
- **首年免年費**。
- 次年免年費條件（任一即可）：
  1. 年消費滿 **NT$30,000** 或 **6 筆消費**。
  2. 申辦電子帳單且任刷一筆消費。
- 信用卡循環年利率：6.75% ～ 15%（依本行信用評分制度定期評估）。
- 預借現金手續費：預借金 × 3% + NT$150（或美元 5 元）。

## 適合情境

- 蝦皮購物高頻買家：站內 10% 蝦幣（4% + 6%）+ 滿額禮 650 / 1,800 蝦幣。
- 月帳單消費 ≥ NT$5,000 或 5 筆：免運券自動回饋。
- 海外消費：免手續費 + 0.5% 蝦幣。
- 經常使用指定外送 / 交通 / 航空 / 旅行社 / 旅遊平台 / 免稅店：登錄後 7% 蝦幣。

## 不適合情境

- 非蝦皮購物用戶：蝦幣折抵僅限蝦皮全站。
- 想要附卡：本卡**無附卡**。
- 想累積多張蝦皮聯名卡：每蝦皮會員 / 持卡人歸戶各限 1 張。
- 主力為現金回饋而非蝦幣：蝦幣僅能折抵蝦皮全站。
- 年消費未達 NT$30,000 或 6 筆且未申辦電子帳單：次年起 NT$1,800 年費。
