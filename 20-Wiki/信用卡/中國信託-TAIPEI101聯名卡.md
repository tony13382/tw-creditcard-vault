# 中國信託 TAIPEI 101 聯名卡

## 定位

中國信託與台北金融大樓股份有限公司聯名 VISA 信用卡。三卡別：**尊榮鼎極卡（邀約制，NT$28,000 年費）/ 鼎極卡（NT$5,000）/ 御璽卡（NT$1,800）**。回饋為 **101 Point**（鼎極 / 尊榮鼎極卡）或現金回饋（御璽 / 鈦金卡），館內最高 3% 無上限、信義區停車禮遇、觀景台與餐券優惠。

## 2026 可用回饋

### 尊榮鼎極卡（邀約制）

| 情境 | 回饋 |
|---|---|
| 館內消費 | **最高 3%**（國內 1% 現金回饋 + 2% 101 Point） |
| 國內一般消費 | 1% 現金回饋 |
| 國外消費 | 1.2% 現金回饋 |
| **國外實體商店登錄加碼**（115/1/1 ～ 115/6/30） | **最高 3%**（1.2% + 加碼 **1.8%**） |

- 加碼 1.8% 上限：**每季歸戶 NT$2,500**；活動期間總回饋上限 NT$500,000；需登錄。
- 加碼分期回饋：115/1/1 ～ 115/3/31 於 115/5/15 起回饋；115/4/1 ～ 115/6/30 於 115/8/14 起回饋。

| 國內一般消費 | 1% 現金回饋無上限 |
| 國外消費 | 1.2% 現金回饋無上限 |

### 御璽卡

| 情境 | 回饋 |

| 每產品當期或前期帳單國內新增消費 | 當期國外消費回饋率（含基本 0.5%） |

| NT$20,000 ～ NT$39,999 | 1% |
| NT$40,000 ～ NT$99,999 | 1.5% |
| NT$100,000 以上 | **2.5%** |

### 已停發舊卡

- **鼎極無限卡 / 鼎極世界卡**：無限卡國內紅利 3 倍 / 國外 5 倍；世界卡 0.8% / 1.2% 現金回饋。
- **御璽卡（舊）/ 鈦金卡（舊）**：館內 3 倍紅利、月帳單滿 NT$3,000 起累積；鈦金卡 0.5% 現金回饋 + 國外消費級距式最高 2.5%。

## 生日禮讚（所有卡別）

生日當月**前 3 個月**（不含寄送當月）館內累計刷卡消費滿額：

| 累計消費 | 餐券 |
|---|---|
| NT$6,000 以上 | 高空景觀餐廳**雙人下午茶**兌換券乙份 |
| NT$60,000 以上 | 高空景觀餐廳**雙人午餐**兌換券乙份 |

> 擇優兌換（高金額擇優）。

## 使用條件

- 申辦資格：
  - **尊榮鼎極卡（邀約制）**：年滿 18 歲，固定年收入 NT$200 萬以上或近 3 個月平均資產 NT$240 萬以上。
  - **鼎極卡**：同上。
  - **御璽卡**：年滿 18 歲，固定年收入 NT$20 萬以上。
- 同一個卡等每人限申請 1 卡。
- 附卡：年滿 15 歲；未滿 18 歲須法定代理人共同簽名同意；申請書正本連同身分證影本寄回（不接受傳真）。
- 申請御璽附卡，正卡人需同時持有相同之御璽卡。
- 學生：年收入需 NT$10 萬以上。
- 國外實體商店定義：請款時消費地或商店登記國別碼非台灣，且須面對面交易；網路、條碼支付、第三方支付不列入。

## 排除條款

- 不回饋交易項目不予計算回饋。
- 國外實體商店加碼活動的加碼商店認定依本行系統為主。
- 退貨或消費退款：原先取得回饋金將依現金回饋比例扣除；不足扣回時列示為應繳金額。
- 違反信用卡約定條款者，本行有權暫停回饋資格與追回回饋金。

## 適合情境

- 高頻台北 101 館內消費者：館內 3% 回饋（鼎極卡含 2% 101 Point + 1% 現金）。
- 經常於信義區購物 / 用餐：4 小時免費停車（尊榮鼎極）或 2 小時（鼎極卡）。
- 想看 101 觀景台：尊榮鼎極 8 折 / 鼎極 9 折優惠。
- 國外旅遊高頻者：登錄後 H1 國外實體 3%（尊榮鼎極）或御璽卡達 NT$100,000 國內月消費後國外 2.5%。
- 生日季能集中 NT$60,000 館內消費換高空餐廳雙人午餐。
- 收入 NT$200 萬以上有資格申辦鼎極卡或受邀尊榮鼎極卡。

## 不適合情境

- 不常去 101：館內 3% 用不到。
- 不在信義區開車：停車優惠用不到。
- 收入未達 NT$200 萬：只能申辦御璽卡，回饋率 0.5% 起跑、需衝高國內月消費才有國外 2.5%。
- 鼎極卡 NT$5,000 年費對非高頻 101 客群成本不划算。
- 尊榮鼎極卡需 101 尊榮俱樂部邀約。
