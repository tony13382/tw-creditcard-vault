# 華南 Rich+ 富家卡

## 定位

華南銀行 Mastercard 鈦金等級的「現金回饋 + 儲蓄點數」雙軌理財型回饋卡。基本國內 1% / 國外 2% 現金回饋無上限為持續性回饋；儲蓄點數加碼依華南理財會員等級（小／大／鉅富家）給 +1% ~ +3%（2026 上半年活動至 2026-06-30），名目最高 國內 4% / 國外 5% 是「鉅富家 + 儲蓄點數能折抵」雙條件成立時的數字。

**關鍵限制**：儲蓄點數 1 點 = NT$1，但要折抵刷卡消費**必須**符合「華南理財扣款」四選一條件（基金定期定額／單筆申購／華南人壽健康年金險／外國債）。對非華南理財客戶，等值回饋只剩基本 1% / 2%。升等大富家 / 鉅富家還需電子帳單與多項華南金融商品持有條件。

## 等級成立條件

- **小富家**：基本等級（未符合升等者皆屬之）。
- **大富家**：電子帳單 + 過去半年使用儲蓄點數 1 次（首次申請者豁免，待官方確認）+ 以下四選二：
  - 近 3 月平均月簽帳 NT$5,000
  - 華南非貨幣型基金或外國債近 3 月餘額 > 0
  - 華南人身保險近 3 月餘額 > 0
  - 華南房貸近 3 月餘額 > 0
- **鉅富家**：電子帳單 + 過去半年使用儲蓄點數 3 次（首次申請者豁免，待官方確認）+ 以下三選二：
  - 華南領航會員獨享家（金額門檻依會員等級為準）
  - 華南人身保險近 3 月餘額 > 0
  - 華南外國債近 3 月餘額 > 0

## 儲蓄點數規則

- **兌換**：1 點 = NT$1，折抵刷卡消費。
- **折抵條件（需符合任一）**：
  - 前月使用 Rich+ 富家卡定期定額扣基金
  - 前月單筆申購基金
  - 新申購華南代理之人壽 / 健康 / 年金險，核保日近 3 個月內未撤銷
  - 近 3 個月新申購外國債
- **效期**：半年為週期，每年 6/30、12/31 清算；逾期未動失效，有動用則展延半年。

## 使用條件

- 申辦資格：正卡須年滿 18 歲、有正當職業；附卡須年滿 15 歲。
- 電子帳單：大富家、鉅富家升等必要條件之一。
- 登錄：基本回饋與儲蓄點數加碼依當月等級自動計算。

## 排除條款

- 特約商店分期付款交易無現金與紅利回饋（但可計入新戶首刷禮消費門檻）。
- 預借現金。
- 第三方支付、加值依華南銀行「一般消費定義」公告為準（待後續補核完整清單）。
- 規費、稅費、保費等依華南通用一般消費定義。

## 適合情境

- 華南理財客戶：每月有基金定期定額、單筆申購、人壽 / 健康 / 年金險或外國債申購，能達成儲蓄點數折抵條件。
- 鉅富家等級者：能吃到國內 4% / 國外 5% headline，且每月消費可達 NT$150,000 - 333,333 範圍內。
- 大富家等級者：能吃到國內 3% / 國外 4%，月消費可達 NT$150,000 範圍內。
- 想用信用卡疊加「投資 + 消費」雙軌的高金額理財客戶。

## 不適合情境

- 非華南理財客戶：等值回饋只剩 1% / 2%，可參考無上限基本卡如 [[20-Wiki/信用卡/上海商銀-簡單卡|上海商銀簡單卡]]（國內 1% / 國外 2% / 日韓 3%）、[[20-Wiki/信用卡/永豐-現金回饋卡|永豐現金回饋卡]]、[[20-Wiki/信用卡/永豐-Me-Card|永豐 Me Card]]。
- 願意理財但不打算把資金集中華南：可能無法穩定觸發儲蓄點數折抵條件。
- 主力為網購、行動支付或第三方支付者：Rich+ 富家卡無這些通路加碼，可參考 [[20-Wiki/信用卡/兆豐-e秒刷鈦金卡|兆豐 e 秒刷鈦金卡]]、[[20-Wiki/信用卡/玉山-U-Bear信用卡|玉山 U Bear 信用卡]]。
- 主力消費為保費 / 稅費 / 規費 / 分期者：基本與儲蓄點數加碼皆不認列。
