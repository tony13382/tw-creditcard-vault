# 高雄銀行金卡 / 高雄市政府認同卡（金卡）

## 定位

高雄銀行基本金卡與認同卡，**共用同一份月累計分級回饋表**（0.1%-0.6% + 單筆 NT$5,000 加值 +0.2%，最高 **0.8%**）。差異僅在卡面圖樣與認同身分。

兩張卡均不具高階卡屬性：旅平 NT$1,500 萬（鈦金卡的一半）；基本回饋率低於市場 1% 基準；**主要適合特定情境**（高雄市民支持公益認同身分 + 偶爾低額消費），不適合做為主力卡。

## 月累計分級回饋（金卡 / 認同卡共用）

| 每月刷卡金額 | 基本回饋 |
|---:|---:|
| 未滿 NT$10,000 | 0.1% |
| NT$10,000 - 19,999 | 0.2% |
| NT$20,000 - 39,999 | 0.3% |
| NT$40,000 - 59,999 | 0.4% |
| NT$60,000 - 79,999 | 0.5% |
| NT$80,000 以上 | **0.6%** |

**單筆消費 NT$5,000 以上 → 該筆額外加 +0.2%**。

實際換算：
- 月刷 NT$80,000、單筆全在 NT$5,000 以上：0.6% + 0.2% = **0.8%**（手冊 headline 最高值）。
- 月刷 NT$50,000、單筆全在 NT$5,000 以上：0.4% + 0.2% = 0.6%。
- 月刷 NT$10,000、單筆全在 NT$5,000 以上：0.2% + 0.2% = 0.4%。

## 年費

- 正卡 NT$2,400 / 附卡 NT$1,200。
- **申請首年免年費，年年有消費即年年免年費**。

## 旅遊保險

需以本卡支付旅遊團費 80% 以上 或 公共交通工具全部票款：

- 旅行平安（傷害）險：**NT$1,500 萬**（鈦金卡為 NT$3,000 萬）。
- 旅平醫療：NT$10 萬。移靈：NT$3 萬。
- 班機/行李延誤：NT$7 千／家屬合計 NT$1 萬 4 千。
- 行李遺失：NT$3 萬／家屬合計 NT$6 萬。
- 行程縮短取消：NT$3 萬／家屬合計 NT$6 萬。

## 道路救援

- 與鈦金卡相同：一般消費達 NT$3,000 → 30km 免費拖吊（全鋒 0800-007-222）。

## 國外交易

- 國外交易服務費 **0.5%**（外加 1% 國際手續費）。
- 無國外加碼；扣除手續費後多數情境淨值為負，不適合海外消費。

## 排除回饋

- 預借現金、稅費款、規費、醫療費、學雜費、公共事業費用。
- 循環利息、各項手續費。
- 公務卡、採購卡不適用。

## 排除條款

- 預借現金、特約商店分期、稅款、規費、政府機關費用、第三方支付（依官方公告）通常不享回饋。
- 各加碼活動依官方公告之有效期間與條件為準，過期後須複查。
- 退貨或爭議款將扣回相應回饋。

## 適合情境

- 高雄市民希望透過認同卡支持公益身分（具體公益捐款比例手冊未獨立列示，待補核）。
- 高雄銀行存戶 + 偶爾旅遊（旅平 NT$1,500 萬保障）。
- 偶爾自駕跨縣市：30km 免費拖吊有實用價值。

## 不適合情境

- **主力消費卡需求**：最高 0.8% 低於市場 1%-1.5% 基準，效率明顯落後。
- 主力海外消費：0.5% 服務費 + 1% 國際手續費 → 多數情境淨值為負。
- 主力稅費、保費、公共事業、學雜費者：完全排除。
- 大量小額消費（單筆 < NT$5,000）：取不到加值 0.2%，月累計需達 NT$80,000 才有 0.6%。
