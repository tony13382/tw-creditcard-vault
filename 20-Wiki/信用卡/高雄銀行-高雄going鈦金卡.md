# 高雄銀行 高雄 going 鈦金卡

## 定位

高雄銀行主力鈦金卡，**單筆刷卡金額分級回饋（0.1%-1.2%）**。網頁 headline「最高 1.2% 無上限」實際需**單筆消費 NT$150,000 以上**才能享。回饋無金額上限但分級綁單筆，需高客單價或大額單筆消費才划算。額外提供 NT$3,000 萬旅平、30km 免費拖吊。

## 分級現金回饋

| 單筆刷卡金額 | 回饋比例 | 換算每筆回饋 |
|---:|---:|---:|
| < NT$1,000 | **無** | — |
| NT$1,000 - NT$2,999 | 0.1% | $1-$3 |
| NT$3,000 - NT$9,999 | 0.3% | $9-$30 |
| NT$10,000 - NT$29,999 | 0.4% | $40-$120 |
| NT$30,000 - NT$49,999 | 0.5% | $150-$250 |
| NT$50,000 - NT$99,999 | 0.7% | $350-$700 |
| NT$100,000 - NT$149,999 | 1.0% | $1,000-$1,500 |
| **NT$150,000 以上** | **1.2%** | $1,800+ |

- 次月以現金回饋方式抵充應收帳款。
- 一般日常消費（單筆 NT$1,000-2,999）僅 0.1%，明顯低於市場基本盤 1%。
- **目標客群**：高客單價（家電、旅遊團費、家具、3C、奢侈品）或固定大額（裝潢、學雜費除外）。

## 年費

- 正卡 NT$2,600 / 附卡 NT$1,300。
- **申請首年免年費，年年有消費即年年免年費**。

## 旅遊保險（鈦金卡專屬高額）

需以本卡支付旅遊團費 **80% 以上** 或 公共交通工具全部票款：

- 旅行平安（傷害）險：**NT$3,000 萬**（含配偶、未滿 25 歲未婚子女）。
- 旅平醫療：NT$10 萬。移靈：NT$3 萬。
- 班機/行李延誤：NT$1 萬／家屬合計 NT$2 萬。
- 行李遺失：NT$3 萬／家屬合計 NT$6 萬。
- 行程縮短取消：NT$3 萬／家屬合計 NT$6 萬。

## 道路救援

- 最近一期一般消費達 **NT$3,000 以上** 即享 24 小時免費急修 + **30km 免費平面拖吊**。
- 全鋒 0800-007-222。

## 國外交易

- **國外交易服務費 0.5%**（外加國際組織 1%）；無國外加碼。
- 國外交易回饋仍依單筆分級表認列；扣除 1.5% 手續費後實際回饋可能為負（單筆 NT$3,000-99,999 為 0.3-0.7%，淨值 -1.2 到 -0.8%）。
- **不適合海外消費主力**。

## 排除條款

- 預借現金、特約商店分期、稅款、規費、政府機關費用、第三方支付（依官方公告）通常不享回饋。
- 各加碼活動依官方公告之有效期間與條件為準，過期後須複查。
- 退貨或爭議款將扣回相應回饋。

## 排除回饋

- 預借現金、稅費款、規費、醫療費、學雜費、公共事業費用。
- 循環利息、各項手續費。
- 公務卡、採購卡不適用。

## 適合情境

- 偏好「**單筆大額**」消費（裝潢、家電、3C、旅遊團費、家具）：單筆 NT$150,000 以上 1.2% 確實高於基準。
- 高雄銀行存戶 + 偶爾出國（旅平 NT$3,000 萬保障佳）。
- 自駕族 + 跨縣市出遊：30km 免費拖吊有實用價值。

## 不適合情境

- **日常小額消費為主**：單筆 < NT$3,000 只有 0.1%-0.3%，遠不如市場 1%-1.5% 基準。
- 主力海外消費：0.5% 服務費 + 1% 國際手續費 - 級距回饋 → 多數情境淨值為負。
- 主力稅費、保費、公共事業、學雜費者：完全排除。
- 不刷大筆者：年費 NT$2,600 在小額用戶下難回本。
