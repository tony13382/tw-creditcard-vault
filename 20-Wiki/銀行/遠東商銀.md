# 遠東商銀

## 公開資料範圍

- 本頁只保留遠東商銀信用卡公開層級規則。
- 個人信用額度、持卡狀態、帳單與繳款、調額紀錄、遠銀 VIP 星等與帳戶自扣狀態不納入公開銀行頁。

## 銀行層級規則

- 同一銀行信用卡通常共用同一信用額度；公開卡片頁只記錄所屬銀行與共用額度規則，不記錄任何實際額度。
- 遠東商銀多張信用卡使用刷卡金與 HAPPY GO 點數作為回饋載體；推薦時需分清楚現金回饋、HAPPY GO 點數與活動贈品。
- 遠東樂行卡多數加碼需當期帳單一般消費滿 NT$3,000、完成活動登錄、一次付清，且交易需符合指定通路或指定活動定義。
- 遠東樂行卡的新辦帳戶自扣 1% 需遠銀 VIP 四星等含以上客戶與新申辦遠銀帳戶自扣；個人是否符合星等或是否已設定自扣由私有層判斷。
- C'est Moi 我的卡主要回饋載體為 HAPPY GO 點數；最高 4 倍 / 3 倍需當期帳單新增一般消費滿 NT$5,000，且單筆每滿 NT$100 才計點。
- C'est Moi 我的卡外幣消費滿額刷卡金、指定地區加碼、高鐵 / 台鐵刷卡金與新戶任務皆需看登錄、滿額級距、排除交易與活動期間。
- 樂家+卡是寵物 / 家庭生活與遠東集團通路導向卡；寵物最高 10% 與指定生活最高 4% 皆需看指定通路、付款方式、遠銀帳戶自扣與上限。
- 樂家+卡遠東集團指定商店回饋為 HAPPY GO 點數；一般消費 0.5% 刷卡金不含已提供 HAPPY GO 點數之消費。
- 快樂信用卡是指定生活、LINE Pay、悠遊卡自動加值與 HAPPY GO 導向卡；指定生活 5% 需登錄、LINE 個人化綁定、限量與月上限，不能當作無條件 5%。
- 保費分期活動與保費刷卡金通常互斥，且分期需信用額度內；公開層不保存個人額度。

## 注意事項

- 推薦遠東樂行卡前需確認一般消費門檻、活動登錄、指定通路、是否第三方支付、入帳月份、國外交易國別、保費是否一次付清或分期。
- 推薦 C'est Moi 我的卡前需確認 HAPPY GO 點數價值、是否為指定百貨 / 旅遊 / 海外消費、當期一般消費門檻、是否使用 HAPPY GO Pay、是否外幣滿額、是否屬歐洲指定國家實體商店。
- 推薦樂家+卡前需確認是否為官方指定寵物通路 / 動物醫院、是否官方網站 / App、是否第三方支付或百貨專櫃、是否已設定遠銀帳戶自扣，以及海外是否為歐洲指定國家實體店。
- 推薦快樂信用卡前需確認指定生活 / LINE Pay 是否符合帳單明細、是否每月登錄、是否仍綁定遠銀 HAPPY+ LINE、是否為百貨型或賣場型門市，以及是否超過月上限。
- 國外消費回饋排除歐洲指定國家實體商店；海外網站與實體交易需分開判斷。
- 新戶活動需確認遠銀信用卡新戶定義、核卡日期與線上申辦來源。
