# 土地銀行

## 公開資料範圍

- 本頁只保留銀行層級公開規則。
- 個人信用額度、持卡狀態、帳單與繳款、調額紀錄不納入公開銀行頁。

## 定位

臺灣土地銀行為公股銀行體系發卡機構，傳統公股卡定位偏保守。卡片組合以 icash 2.0 聯名鈦金卡（國內 1% / 國外 2.5% 為公股中相對少見的海外回饋）為主力，搭配 JCB 系列（極緻 / 晶緻 / 一卡通聯名晶緻）、白金商務 / 國際信用卡 / 康鉑系列等傳統公股卡。特色為**房貸 / 消貸客戶專用版**信用卡，是少數針對借貸客戶量身設計卡別的銀行。

## 銀行層級規則

### 信用額度

- 土地銀行信用卡通常共用同一銀行信用額度。
- 個人實際信用額度、調額紀錄：不納入公開銀行頁。

### 紅利 / 現金回饋機制

- 主要採**現金回饋**與**JCB 紅利點數**併行。
- icash 2.0 鈦金卡回饋以現金回饋為主，於次期帳單折抵。
- JCB 系列卡享 JCB 國際品牌活動。

### 帳單方式

- 電子帳單為部分加碼級距條件。
- 土銀存款帳戶自扣為部分卡別優惠條件。

### 排除條款（共通）

- 預借現金、分期付款、循環信用、年費、規費（含稅費、罰金、交通違規罰款）、保費等通常不計入回饋。
- 房貸 / 消貸專用卡可能限該類客戶申辦，非一般客戶。

## 銀行特殊規則

- **房貸 / 消貸客戶專用版卡**：白金商務卡、國際信用卡、康鉑晶片卡均有「一般 / 房貸 / 消貸」三版本，為土銀特色卡別設計。
- **JCB 極緻卡**：頂級卡，年費 NT$10,000，僅高資產客戶適合。
- **2026 繳稅活動**：稅額 NT$5,000 以上享 3/6/9 期 0 利率（全卡別適用）。
