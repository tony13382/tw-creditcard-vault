# 元大銀行

## 公開資料範圍

- 本頁只保留銀行層級公開規則。
- 個人信用額度、持卡狀態、帳單與繳款、調額紀錄不納入公開銀行頁。

## 銀行層級規則

- 元大商業銀行信用卡共用同一信用額度。
- 鑽金紅利系統：1 點 = NT$1，作為「現金回饋」入帳；非傳統紅利點數。
- 信用卡循環年利率：2.92%~15.00%（基準日 2017-04-01）。
- 每筆預借現金手續費：金額 × 3.5% + NT$100。
- 元大銀行帳戶自動轉帳扣繳信用卡款 與 信用卡電子帳單 是常見的年費減免與禮遇門檻條件。
- 一般消費定義公告完整排除清單本次未獨立查核，後續批次補核。
- 元大鑽金卡與元大鑽金**智富**卡是不同產品；智富卡為 icash 聯名版，2022-2023 年曾辦理「外幣帳戶綁定 + 自扣 + 新戶」海外加碼活動（最高 4% / 6%），活動已過期。

## 公開帳戶條件

- 自動扣繳規則：元大帳戶自動轉帳扣繳元大信用卡款；於扣繳期間享部分卡片免年費（鑽金卡為例）。
- 電子帳單規則：申辦電子帳單於該期間享部分卡片免年費（鑽金卡為例）。

## 分期或大額消費規則

- 分期相關規則本次未獨立查核。
- 公開卡片頁不記錄個人信用額度與分期狀態。

## 注意事項

- 不保存完整卡號、帳號、身分證字號或客服驗證資訊。
- 推薦元大鑽金卡前需確認：是否需禮遇（觸發條件為前月或前 3 個月消費門檻）、是否要免年費（電子帳單／自扣）。
- 元大銀行官網以 yuantabank.com.tw 為主域；產品頁採查詢字串 ID 形式（`/bank/creditCard/creditCard/in.do?id=<id>`）；信用卡入口頁與其他子頁本次部分回 HTTP 500，需後續以 playwright + chromium 渲染抓取。
