# 永豐聯名通路型卡片比較

## 結論

永豐 P3 聯名通路型卡要先看「通路或會員身份」而不是回饋率。多數卡只有在特定合作通路、合作方會員制度或卡別權益成立時才有價值。

| 類型 | 卡片 | 推薦定位 |
|---|---|---|
| 百貨/影城通路 | [[20-Wiki/信用卡/永豐-美麗華卡|永豐美麗華卡]] | 常去美麗華百樂園或大直影城，且會用電影刷卡金、館內紅利、停車與分期時才評估 |
| 合作方點數 | [[20-Wiki/信用卡/永豐-東森網連通聯名卡|東森網連通聯名卡]] | 需東森網連通加盟店主或實習店主；PV 價值不能直接用現金回饋排序 |
| 多卡別會員制 | [[20-Wiki/信用卡/永豐-美安聯名卡|美安聯名卡]] | 需拆卡別、會員/店主/法人資格、SHOP.COM、IBV、運動、國外消費與高階權益 |
| 低回饋通路卡 | [[20-Wiki/信用卡/永豐-美樂家生活卡|美樂家生活卡]] | 美樂家館內 0.6%、一般 0.4%；只適合美樂家常用者 |
| 一般/保費補位 | [[20-Wiki/信用卡/永豐-VOGUE卡|VOGUE 卡]] | 最高 1.2% 細則需複查；保費 1% 需注意排除保險公司與月上限 |

## 推薦判斷

1. 先問使用者是否真的使用聯名通路：美麗華、美安、美樂家、東森網連通或 VOGUE 卡別權益。
2. 若是合作方點數或回饋，先確認是否有會員/店主資格與點數價值：東森 PV、美安 IBV、美安現金回饋。
3. 若是美安，先拆卡別：運動一卡通、悠遊鈦金、悠遊無限、鈦金商務公司卡差異很大。
4. 若只是一般消費，優先回到 [[20-Wiki/策略/永豐現金回饋系卡片比較|永豐現金回饋系卡片比較]]，不要用聯名卡當主力。
5. 若是保費，查 [[20-Wiki/通路/保費|保費]]，不要把一般消費 headline 套到保費。

## 風險

- 美麗華多數權益公告至 2026-06-30，且店櫃、現場公告與行動支付限制多。
- 東森 PV 與美安 IBV / 現金回饋由合作方提供，合作方制度變更會影響實際價值。
- 東森網連通聯名卡與美安聯名卡會固定結帳日為每月 3 日。
- 美安活動同頁包含多卡別，且有上半年加碼與全年權益混在一起，推薦前需拆活動期間。
- VOGUE 最高 1.2% 細節在文字擷取中不完整，需複查原頁完整表格；保費文案與注意事項的上限表述不完全一致。
