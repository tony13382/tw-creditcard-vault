# 聯邦銀行研究

## 定位

## 仍可申辦卡片總覽（11 張個人卡）

依官方 2026-05-31 盤查：

### 銀行卡

- **M 卡**：旅遊交通五大場景 + 11% 跨場景
- **賴點卡**：LINE Pay 11% + 國外 3% + 歐洲 3%
- **吉鶴卡**：日本旅遊祭 11% + 11 大商店 + 日系名店 5.5%
- **綠卡**：ESG 綠色消費 6%
- **理財型白金卡**：每月 NT$450 / 8% 回饋
- **幸福御守卡**：國內 0.5% / 國外 1.5%

### 頂級卡

- **M 世界卡**：年收 NT$100 萬以上；六大場景 11% + 高階旅遊權益（年費 NT$3,600 無減免）

### 聯名卡

- **LINE Bank 聯名卡**：純網銀聯名
- **MaiCoin 聯名卡**：加密貨幣聯名（**台灣首張**）
- **微風聯名卡**：百貨卡，平日免費停車 4 小時
- **全國加油聯名卡**：加油卡，每人限 1 張

## 待研究方向

- 聯邦無限卡（高階卡）：活動頁有「2026 聯邦無限卡」（https://activity.ubot.com.tw/aws_act/2026/ubinfinite/index.htm）需進一步檢視。
- 聯邦國民旅遊卡：屬國民旅遊卡專案類，依公務員適用。
- 各卡 H2 加碼期間與 Q3/Q4 活動延續性需依各活動頁年度複查。
- 跨銀行比較：
  - **LINE Pay 卡**：聯邦賴點卡（11%）vs 中信 LINE Pay 卡 vs 台北富邦 J 卡 LINE Pay 10%。
  - **日本旅遊卡**：聯邦吉鶴卡（11%）vs 玉山熊本熊（8.5%）vs 富邦 J 卡（6%）。
  - **綠色卡**：聯邦綠卡（6%）vs 永豐 Green 卡（5%）vs 星展 eco 永續卡（10%）。
  - **加密貨幣卡**：聯邦 MaiCoin（台灣唯一）。

## 整理策略

1. ✅ 完成銀行卡批次：M 卡 + 賴點卡 + 吉鶴卡 + 綠卡 + 理財型白金卡 + 幸福御守卡。
2. ✅ 完成頂級卡批次：M 世界卡。
3. ✅ 完成聯名卡批次：LINE Bank 聯名卡 + MaiCoin 聯名卡 + 其他卡群索引（微風 + 全國加油）。
4. 後續：聯邦無限卡（高階）、國民旅遊卡（公務員用）。
