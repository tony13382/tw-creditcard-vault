## 整理策略

1. 鑽金卡已升級公開卡片頁；2026 全年國內 1.2% / 海外 2.2% 無上限無門檻；元大銀行公開規則頁同步補強。
2. 元大網站為 SPA + .do servlet，WebFetch 直接讀回 HTTP 500 / 空殼；本批改以本地 playwright + chromium 渲染後抓取 tab 內容（quest/answer 結構），能完整取得官方資料。
3. 第三方提及「海外 4% / 6%」是 2022-2023 鑽金智富卡（另一張，icash 聯名版）已過期活動，與 2026 鑽金卡無關；正式推薦時需明確區分兩張卡。
4. 下一輪可優先處理鑽金智富卡與元大銀行卡（鑽金 Visa 簽帳卡、Visa 鑽鑽卡、Visa 鑽金卡等可能變體）。
