## 整理策略

1. 補核流程：以 Chrome 直連 fubon.com 主站逐 tab 抓取；類似 CTBC、國泰流程。
2. P1 批次先做 3 張銀行卡（尊御世界、鑽保、富利生活），可分別形成跨銀行策略樣本（高階卡、保費卡、生活通路卡）。
3. P2 批次處理 4 張聯名卡（悍將勇士 / 台茂 / 采盟 / 福華），其中悍將勇士可作為運動聯名卡跨銀行比較。
4. 認同卡群（Angel / 法鼓山 / 台北市政府）合併索引頁（類似 [[20-Wiki/信用卡/中國信託-認同卡群]] 模板）。
5. 富邦長年大規模停發舊卡（鈦金 / 世界 / 無限 / 銀行卡 / 商務 / a miles / 財神 / 數位生活 / Amoney 等），可寫入 [[20-Wiki/銀行/台北富邦銀行]] 的「卡別演變」段落。
6. 不寫入個人持卡、共用額度的實際數字、調額紀錄；台北富邦共用額度公開規則只放 [[20-Wiki/銀行/台北富邦銀行]]。
