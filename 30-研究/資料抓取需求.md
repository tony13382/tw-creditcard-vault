# 資料抓取需求

這層用來管理「還沒整理成 wiki 的公開信用卡研究需求」。研究清單只代表可研究或待整理，不代表使用者持有。

## 研究分層

| 層級 | 目的 | 產出 |
|---|---|---|
| R0 發卡機構母清單 | 確認台灣有哪些信用卡發卡機構值得研究 | [[30-研究/發卡機構清單|發卡機構清單]] |
| R1 銀行卡片索引 | 逐銀行抓官方卡片總覽，列出所有可研究卡片 | [[30-研究/信用卡研究清單|信用卡研究清單]]、`30-研究/銀行/` |
| R1b 熱門候選池 | 用第三方榜單、官方統計與社群搜尋訊號決定研究優先順序 | [[30-研究/熱門信用卡候選清單|熱門信用卡候選清單]] |
| R2 單卡資料抓取 | 對候選卡抓產品頁、活動頁、條款、回饋上限與排除條款 | `10-來源/官方活動/` 或 `10-來源/條款/` |
| R3 Wiki 整理 | 只有資料夠完整、會被推薦或比較時，升級為公開卡片頁 | `20-Wiki/信用卡/` |
| R4 個人化套用 | 不屬於公開 repository；公開資料只提供可查核規則 | 不納入 git |

## 抓取欄位

| 欄位 | 說明 | 必要性 |
|---|---|---|
| 發卡機構 | 正式金融機構名稱 | 必填 |
| 卡片正式名稱 | 官方卡名 | 必填 |
| 卡片別名 | 常見簡稱、舊名、產品整併名稱 | 建議 |
| 官方產品頁 | 官方卡片介紹 URL | 必填 |
| 官方活動頁 | 回饋活動、首刷禮、期間加碼 URL | 必填，若無則註明未見 |
| 條款或 PDF | 申請書、約定條款、回饋計劃、注意事項 | 必填，若無則註明未見 |
| 卡片分類 | 銀行卡、聯名卡、認同卡、商務卡、雙幣卡、無限/世界卡等 | 必填 |
| 申辦狀態 | 可線上申辦、下載申請書、僅存續、疑似停發、需客服確認 | 必填 |
| 回饋形式 | 現金回饋、點數、LINE POINTS、刷卡金、哩程等 | 必填 |
| 核心回饋 | 國內、國外、指定通路、保費、分期等 | 必填 |
| 回饋上限 | 月上限、期上限、年上限、名額或登錄限制 | 必填 |
| 成立條件 | 自扣、電子帳單、會員等級、App 方案、登錄、領券、新戶資格 | 必填 |
| 排除條款 | 不回饋交易、第三方支付、代收、分期、保費、稅費等 | 必填 |
| 有效期間 | 活動起訖日期；長期權益也要記查核日期 | 必填 |
| 需複查日期 | 通常抓活動結束前兩週，或來源更新週期 | 必填 |
| 研究狀態 | 未開始、已列索引、已抓產品頁、已抓條款、已建 wiki、暫緩 | 必填 |
| 是否已建 wiki | 連到 `20-Wiki/信用卡/`，沒有就留空 | 必填 |

## 升級規則

- 只有卡片名稱出現在官方總覽時，先留在研究清單。
- 有官方產品頁與基本回饋資訊，才可建立來源筆記。
- 有回饋、上限、有效期間、排除條款與成立條件，才可升級成 `20-Wiki/信用卡/`。
- 第三方熱門度只決定研究優先順序；不作正式推薦依據。
- 高階卡、商務卡、企業卡、簽帳金融卡、現金儲值卡先標記分類，不混入一般個人信用卡推薦。
- 個人是否持有、信用額度、帳戶等級、申辦日期、核卡日期，一律不寫在 `30-研究/`。
