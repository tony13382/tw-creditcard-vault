# 2026-05-30 高雄銀行 高雄 going 鈦金卡回饋查核

- 官方信用卡權益手冊（115.01 版，民國 115/2026 年 1 月版）：https://www.bok.com.tw/card-apply-4

- 需複查日期：2026-12-15

## 查核結論

**重要修正**：高雄 going 鈦金卡是**單筆刷卡金額分級回饋**，「最高 1.2%」需單筆消費達 **NT$150,000 以上** 才能享有。非單筆達門檻不取得對應級距回饋；前次官方頁 headline 對外宣稱「最高 1.2% 無上限」屬於最高級距而非全卡通用。

## 回饋分級表

| 單筆刷卡金額 | 回饋比例 |
|---:|---:|
| NT$1,000 - NT$2,999 | 0.1% |
| NT$3,000 - NT$9,999 | 0.3% |
| NT$10,000 - NT$29,999 | 0.4% |
| NT$30,000 - NT$49,999 | 0.5% |
| NT$50,000 - NT$99,999 | 0.7% |
| NT$100,000 - NT$149,999 | 1.0% |
| **NT$150,000 以上** | **1.2%** |
| NT$1,000 以下 | 不回饋 |

- 「無上限」指的是回饋無金額上限，但**單筆級距以外金額不能合計湊**：必須單筆達 NT$1,000 起跳。
- 本月消費**次月以現金回饋方式抵充應收帳款**。

## 排除回饋

- 預借現金、各項稅費款、規費、醫療費、學雜費、公共事業費用。
- 循環信用利息、各項手續費。
- 公務卡與採購卡不適用。

## 喪失回饋資格

- 未按時繳付帳單最低應繳金額。
- 信用卡被本行強制停卡或自行取消。
- 已為本行催收帳戶或信用管制戶。

## 年費

- 正卡：**NT$2,600** 元
- 附卡：NT$1,300 元
- **申請免年費／年年有消費／年年免年費**（首年免年費，次年起當年度有消費即免次年年費）。

## 旅遊保險

需以本卡支付旅遊團費 80% 以上 或 公共交通工具全部票款，即享下列保障（含持卡人、配偶、未滿 25 歲未婚子女）：

- **旅行平安（傷害）保險：NT$3,000 萬**
- 旅平醫療保險金：NT$10 萬
- 移靈費用：NT$3 萬
- 班機延誤：NT$1 萬／家屬合計 NT$2 萬
- 行李延誤：NT$1 萬／家屬合計 NT$2 萬
- 行李遺失：NT$3 萬／家屬合計 NT$6 萬
- 旅行文件重置：NT$5 千／家屬合計 NT$1 萬
- 行程縮短或取消：NT$3 萬／家屬合計 NT$6 萬
- 劫機補償：NT$5 千／每日

## 道路救援（24 小時）

- 最近一期帳單一般消費達 NT$3,000 以上即享免費急修（接電、換胎充氣、代送汽油或送水等）+ **30 公里免費平面拖吊**。
- 全鋒道路救援服務專線：0800-007-222

## 其他費用

- 國外交易服務費：交易金額 **0.5%**（外加各國際組織 1% 手續費）。
- 預借現金手續費：每筆 NT$150 + 預借金額 3%。
- 循環信用年利率：8.541% - 14.965%（按季評定分級）。
- 信用卡掛失費：NT$200／每卡。
- 分期付款手續費：NT$100／每筆。
- 信用卡繳稅費手續費：行照規費／交通罰鍰／定期稅款每筆 NT$20、中華電信費每筆 NT$10、汽車燃料費每筆 1%。
